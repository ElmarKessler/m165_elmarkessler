### Skript zum Erstellen der zwei neuen User
```
use BuchClubDB
db.createUser({
    user: 'readerUser',
    pwd: 'password123',
    roles: [
        { role: 'read', db: 'BuchClubDB' }
    ]
})
```
```
use admin
db.createUser({
    user: 'readWriteUser',
    pwd: 'password456',
    roles: [
        { role: 'readWrite', db: 'BuchClubDB' }
    ]
})
```
